**📂 Repository Contents & Task Details**

**1. Manual Testing: Swag Labs (Sauce Demo)**

**Comprehensive manual testing was performed on the **[Swag Labs](https://www.saucedemo.com/) application to ensure functional correctness and usability**.**

* **Test Plan:** A detailed strategy covering **In-Scope** features like Sign-up, Login, Forget Password, and Delete Account**.**
* **Test Scenarios & Cases:** Developed test cases (e.g., T01-T09) using techniques like **Equivalence Partitioning** and **Boundary Value Analysis** to test valid/invalid credentials and empty fields**.**
* **Bug Reporting:** Identified a **High-Priority/High-Severity** defect (BR-001) where the system allowed login with unauthorized credentials**.**
* **Summary Reports:** Includes a **Test Summary Report** (28/28 cases passed) and a **Bug Summary Report** documenting Medium and Low severity open bugs**.**

**2. Performance & Load Testing: Daraz**

**Conducted performance validation for the ****Daraz** Home and Registration pages to identify scalability limitations**.**

* **Scope:** Measured system behavior under concurrent hits ranging from 50 to 5,000**.**
* **Key Finding:** The system performed reliably up to  **1,000 concurrent hits** **, but generated error messages at ** **5,000 hits** **, indicating a scalability threshold**.

**3. API Testing: FakeStoreAPI (Postman)**

*Note: This task was completed as part of the course training.*

* **Implementation:** Developed a full collection for the **FakeStoreAPI** utilizing the following methods:
  * **POST:** Create new product data.
  * **GET:** Retrieve product details.
  * **PUT/PATCH:** Update existing product information.
  * **DELETE:** Remove products from the database.

**4. Security & AI-Driven Testing**

* **OWASP ZAP:** Performed security vulnerability assessments to identify potential system threats**.**
* **Generative AI Automation:** Explored modern testing landscapes using **AI-based automation tools** (TestRigor) and **Prompt Engineering** to streamline website verification**.**
